# Pemesanan Barang (Dengan Login)

## Akun Default
- Admin: username=admin, password=12345
- Kasir: username=kasir, password=kasir123

## Cara Pakai
1. Jalankan `admin.py` → login sebagai admin untuk menambah / hapus produk.
2. Jalankan `kasir.py` → login sebagai kasir untuk melakukan transaksi.
3. Produk tersimpan di `products.json` dan akun di `users.json`.

Tidak menggunakan database, hanya file JSON.
